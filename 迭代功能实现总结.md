## ✅ 已完成的 MCP 工具服务

### 文件位置

### 已实现的 14 个迭代管理工具

| 工具 | 功能 | 参数 |
|------|------|------|
| `iteration_create` | 创建新迭代 | title, project_id, start_date, end_date, description, goal |
| `iteration_query` | 查询迭代 | project_id, status, search, limit, offset |
| `iteration_get` | 获取迭代详情 | iteration_id |
| `iteration_list` | 获取迭代列表 | project_id, include_stats, limit |
| `iteration_overview` | 迭代概览（统计） | iteration_id |
| `iteration_stats` | 迭代统计数据 | iteration_id, include_trend |
| `iteration_workitems` | 查询工作项 | iteration_id, type, status, assignee_id, limit |
| `iteration_workitem_stats` | 工作项统计 | iteration_id, group_by |
| `iteration_plan` | 迭代规划 | iteration_id, workitem_ids |
| `iteration_add_workitem` | 添加工作项 | iteration_id, workitem_id, workitem_type |
| `iteration_remove_workitem` | 移除工作项 | iteration_id, workitem_id |
| `iteration_update` | 更新迭代 | iteration_id, title, description, goal, start_date, end_date, status |
| `iteration_delete` | 删除迭代 | iteration_id |

## 🔧 使用示例

### 创建迭代
```javascript
{
  "action": "iteration_create",
  "params": {
    "title": "Sprint 1",
    "project_id": "proj-uxin",
    "start_date": "2026-03-01",
    "end_date": "2026-03-15",
    "description": "第一个迭代",
    "goal": ["完成登录功能", "完成需求管理"]
  }
}
```

### 查询迭代列表
```javascript
{
  "action": "iteration_list",
  "params": {
    "project_id": "proj-uxin",
    "include_stats": true,
    "limit": 10
  }
}
```

### 获取迭代概览
```javascript
{
  "action": "iteration_overview",
  "params": {
    "iteration_id": "iter-xxx"
  }
}
```

### 查询迭代工作项
```javascript
{
  "action": "iteration_workitems",
  "params": {
    "iteration_id": "iter-xxx",
    "type": "all",
    "limit": 50
  }
}
```
