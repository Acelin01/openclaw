## 🎯 使用示例

### 1. 创建迭代
```javascript
{
  "action": "iteration_create",
  "params": {
    "title": "Sprint 1",
    "project_id": "proj-uxin",
    "start_date": "2026-03-01",
    "end_date": "2026-03-15",
    "description": "第一个迭代",
    "goal": ["完成登录功能", "完成需求管理"]
  }
}
```

### 2. 查询迭代列表
```javascript
{
  "action": "iteration_list",
  "params": {
    "project_id": "proj-uxin",
    "include_stats": true,
    "limit": 10
  }
}
```

### 3. 获取迭代概览
```javascript
{
  "action": "iteration_overview",
  "params": {
    "iteration_id": "iter-xxx"
  }
}
```

### 4. 查询迭代工作项
```javascript
{
  "action": "iteration_workitems",
  "params": {
    "iteration_id": "iter-xxx",
    "type": "all",
    "limit": 50
  }
}
```

### 5. 显示迭代列表 Artifact
```typescript
viewer.content = {
  kind: "iteration-list",
  data: {
    iterations: [...],
    selectedId: "iter-xxx",
    showActions: true
  }
};
```

### 6. 显示迭代工作项 Artifact
```typescript
viewer.content = {
  kind: "iteration-workitems",
  data: {
    workitems: [...],
    viewMode: "list"
  }
};
```
