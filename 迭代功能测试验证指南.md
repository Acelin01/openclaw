## 🔧 故障排查

### 如果技能调用失败

1. **检查 API 服务是否运行**
```bash
curl -s http://localhost:8000/api/v1/projects | python3 -c "import sys,json; d=json.load(sys.stdin); print('API 状态:', '正常' if d.get('success') else '异常')"
```

2. **检查 Token 是否正确**
```bash
# 测试 Token 是否有效
curl -s http://localhost:8000/api/v1/projects -H "Authorization: Bearer test-api-token-12345" | head -5
```

3. **检查工具是否注册**
```bash
cd /Users/acelin/Documents/Next/AIGC/openclaw/apps/api
npx tsx -e "import {toolRegistry} from './src/lib/ai/tools/mcp/registry.js'; console.log(toolRegistry.getAllTools().filter(t=>t.name.startsWith('iteration_')).map(t=>t.name));"
```

### 如果 Artifact 不显示

1. **检查预览页面是否可访问**
```bash
curl -I http://localhost:3000/iteration-create-preview.html
```

2. **检查 iframe 是否加载**
- 打开浏览器开发者工具 (F12)
- 查看 Console 是否有错误
- 查看 Network 请求状态
