## 🧪 测试步骤

### 步骤 1: 发送消息创建迭代
**操作**: 点击 "测试创建迭代" 按钮

**用户输入**:
```
创建一个新迭代：Sprint 1，时间从 2026-03-01 到 2026-03-15
```

**预期**: 显示用户消息内容

---

### 步骤 2: 调用 iteration_create 技能
**操作**: 点击 "测试技能调用" 按钮

**请求**:
```json
{
  "action": "iteration_create",
  "params": {
    "title": "Sprint 1",
    "project_id": "proj-uxin",
    "start_date": "2026-03-01",
    "end_date": "2026-03-15",
    "description": "第一个迭代",
    "goal": ["完成用户登录功能", "完成需求管理模块"]
  }
}
```

**预期响应**:
```json
{
  "success": true,
  "data": {
    "id": "iter-xxx",
    "title": "Sprint 1",
    "status": "PLANNING"
  }
}
```

---

### 步骤 3: 显示新建迭代 artifact
**操作**: 点击 "显示组件" 按钮

**预期**: 显示新建迭代表单组件（iframe 嵌入预览页面）

---
