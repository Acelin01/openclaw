## 📊 功能对比

### 桌面端 vs 移动端

| 功能 | 桌面端 | 移动端 |
|------|--------|--------|
| 弹框样式 | 居中模态框 | 抽屉式（底部滑出） |
| 表单布局 | 双栏（编辑器 + 侧边栏） | 单栏卡片式 |
| 选择器 | 下拉菜单 | 底部弹框 |
| 按钮尺寸 | 标准 | 加大（触摸友好） |
| 工具栏 | 完整显示 | 横向滚动 |
| 安全区域 | 无 | 适配刘海屏 |
