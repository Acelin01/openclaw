## 🔧 如果无法访问

### 检查服务器
```bash
# 查看服务器进程
ps aux | grep "http.server 3000"

# 应该看到类似输出:
# acelin  47580  0.0  0.3  ...  Python -m http.server 3000
```

### 重启服务器
```bash
# 停止旧服务器
pkill -9 -f "http.server"

# 启动新服务器
cd /Users/acelin/Documents/Next/AIGC/openclaw/apps/chat-lite/dist
python3 -m http.server 3000 &

# 等待 3 秒
sleep 3

# 验证
curl -sI http://localhost:3000/ | head -1
# 应该输出：HTTP/1.0 200 OK
```

### 检查文件
```bash
ls -lh /Users/acelin/Documents/Next/AIGC/openclaw/apps/chat-lite/dist/*.html

# 应该看到:
# test-requirement-list.html    (3.9K)
# test-testcase-artifact.html   (4.1K)
```
