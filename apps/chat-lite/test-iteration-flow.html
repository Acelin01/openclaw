🔄 迭代功能完整测试流程

1️⃣ 发送消息创建迭代

用户输入消息

用户在 chat-lite 聊天框中输入创建迭代的指令

"创建一个新迭代：Sprint 1，时间从 2026-03-01 到 2026-03-15"

2️⃣ 调用 iteration_create 技能服务

MCP 工具调用

chat-lite 调用 uxin-mcp 的 iteration_create 工具

{ "action": "iteration_create", "params": { "title": "Sprint 1", "project_id": "proj-uxin", "start_date": "2026-03-01", "end_date": "2026-03-15", "description": "第一个迭代", "goal": ["完成用户登录功能", "完成需求管理模块"] } }
待测试

3️⃣ 显示新建迭代 Artifact 组件

Artifact 组件展示

在聊天框右侧显示新建迭代表单组件

viewer.content = { kind: "iteration-create", data: { projectId: "proj-uxin" } }

📦 新建迭代组件预览

4️⃣ 查询迭代列表

调用 iteration_list 技能

查询项目下的所有迭代列表

{ "action": "iteration_list", "params": { "project_id": "proj-uxin", "include_stats": true } }
待测试

5️⃣ 显示迭代概览 Artifact 组件

Artifact 组件展示

显示带左侧迭代列表的概览组件

viewer.content = { kind: "iteration-overview", data: { iterationId: "iter-xxx", overview: {...} } }

📊 迭代概览组件预览

6️⃣ 查询迭代工作项

调用 iteration_workitems 技能

查询迭代中的所有工作项（需求/任务/缺陷）

{ "action": "iteration_workitems", "params": { "iteration_id": "iter-xxx", "type": "all" } }
待测试

7️⃣ 显示迭代工作项 Artifact 组件

Artifact 组件展示

显示工作项列表组件

viewer.content = { kind: "iteration-workitems", data: { iterationId: "iter-xxx", workitems: [...] } }

📝 迭代工作项组件预览

8️⃣ 规划迭代

调用 iteration_plan 技能

将工作项规划到迭代中

{ "action": "iteration_plan", "params": { "iteration_id": "iter-xxx", "requirement_ids": ["req-1", "req-2"], "task_ids": ["task-1"] } }
待测试

9️⃣ 显示规划迭代 Artifact 组件

Artifact 组件展示

显示左右两栏规划界面

viewer.content = { kind: "iteration-plan", data: { iterationId: "iter-xxx", availableWorkitems: [...], assignedWorkitems: [...] } }

📋 规划迭代组件预览
