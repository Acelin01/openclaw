## 功能说明

### 1. 整体布局
- **弹框形式**：与之前迭代概览风格一致，包含标题、标签页、内容区和底部按钮
- **标签页**：支持“概览”和“用例”切换（用例页为占位）
- **数据填充**：严格按照图片中的示例数据填充

### 2. 概览内容模块
- **计划信息卡片**：展示计划名称、关联项目、关联迭代、版本
- **执行状态卡片**：
  - 四个统计块：已通过(0)、未通过(0)、暂缓(0)、待测试(1)
  - 进度环形图（100%样式）及总用例数、完成数、完成率
- **计划人员**：
  - 管理员：0%进度条
  - lenny54010...：0%进度条
- **执行人员表格**：
  - 包含用例执行人、分配用例数量、已测数量、未通过、暂缓、缺陷数量、通过率
  - 一行数据：lenny540101@163.com，分配1，已测0，通过率0%
- **相关缺陷**：暂无数据占位

### 3. 交互
- 点击“打开测试计划概览”按钮显示弹框
- 点击遮罩层或关闭按钮可关闭
- 标签页切换（概览/用例）
- 底部保存和关闭按钮

### 4. 设计风格
- 采用与迭代概览相同的配色（主色#52c41a）
- 卡片式布局，圆角边框，柔和阴影
- 表格和进度条样式统一
- 响应式设计，适配移动端

此弹框可直接嵌入到项目管理系统中，方便查看测试计划详情。
